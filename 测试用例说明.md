# 助教招聘系统测试说明

## 1. 文档说明
本文件为第013组《助教招聘系统》项目的测试说明摘要。  
详细测试内容见同目录下的《测试用例文档.docx》。

## 2. 测试范围
本次测试主要覆盖以下模块：

- 登录认证模块
- 角色主页与权限控制模块
- 岗位申请模块
- 个人中心模块
- 简历上传、提取与下载模块
- 论坛与社交模块
- 数据存储与持久化模块

## 3. 测试环境
- 操作系统：Windows / macOS
- 浏览器：Chrome
- 开发语言：Java
- Web 技术：Servlet / JSP / CSS
- 构建工具：Maven
- 运行环境：Tomcat
- 数据存储方式：JSON / 文本文件

## 4. 重点测试内容
### 登录认证
- 正确账号密码登录
- 错误密码登录
- 空用户名或空密码校验
- 未登录访问受保护页面拦截

### 岗位申请
- 查看岗位列表
- 正常申请岗位
- 重复申请校验
- 未登录申请拦截
- 查看申请人列表

### 个人中心
- 查看个人资料
- 修改资料
- 必填项校验
- 未登录访问拦截

### 简历模块
- 上传合法简历
- 上传空文件
- 上传非法格式文件
- 下载简历
- 提取简历文本
